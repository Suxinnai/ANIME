# 为什么选择 Astro？

Astro 是一个旨在使网站构建速度更快的 Web 框架。它通过尽量减少发送到浏览器的 JavaScript 数量来实现这一点。

## 关键特性

- **群岛架构**: 仅在需要时加载交互式组件。
- **零 JS 默认**: 默认情况下不向客户端发送 JavaScript。
- **自带组件**: 使用 React, Preact, Svelte, Vue 等。
